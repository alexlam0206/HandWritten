# HandWritten ✉️
This is the website of HandWritten, a Hack Club YSWS (You Ship, We ~~Ship~~ Sign) project.
And it is a platform that allows Hack Club members to write their Hack Club journey, on paper.

## Quick Start

```bash
npm install
npm run dev
```

## Built With

- **Next.js** - React Framework
- **Theme UI** - Styling with Hack Club Theme
- **Wrangler** - Deployment to Cloudflare Pages

## Deploy

```bash
npm run deploy
```
